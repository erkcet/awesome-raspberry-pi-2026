# Contributing

## Entry Rules

- Must be directly relevant to Raspberry Pi hardware/software ecosystem.
- Must have a clear homepage or repository with working documentation.
- Prefer actively maintained projects (recent activity in last 24 months).
- Avoid duplicate entries.
- Add concise, factual descriptions.

## Pull Request Checklist

- Add link in the appropriate section.
- Keep alphabetical order inside each section when possible.
- Run `scripts/audit-links.sh README.md` and include any failing links notes.
